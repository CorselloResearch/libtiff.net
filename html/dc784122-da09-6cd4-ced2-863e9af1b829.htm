Remarks

            A valid strip number is always returned; out-of-range coordinate values are clamped to
            the bounds of the image. The row parameter is always used in
            calculating a strip. The plane parameter is used only if data are
            organized in separate planes
            (PLANARCONFIG = PlanarConfig.SEPARATE).
