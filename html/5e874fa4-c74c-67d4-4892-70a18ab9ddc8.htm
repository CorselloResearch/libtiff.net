Remarks

            The format is a composite format string that uses the same format as
            Format  method. The method parameter, if
            not null, is printed before the message; it typically is used to identify the
            method in which an error is detected.
            
Applications that desire to capture control in the event of an error should use
            SetErrorHandler(TiffErrorHandler) to override the default error and warning handler.
