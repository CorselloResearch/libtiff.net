Remarks

            This method can be used to read TIFF data from sources other than file. When custom
            stream class derived from TiffStream is used it is possible to read (or
            write) TIFF data that reside in memory, database, etc.
            
Please note, that name is an arbitrary string used as
            ID for the created Tiff. It's not required to be a file name or anything
            meaningful at all.

            Please read remarks for Open(String, String) method for the list of option flags that
            can be specified in mode parameter.
